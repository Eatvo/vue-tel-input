## Development

Clone the project

```bash
  git clone https://github.com/Eatvo/vue-tel-input.git
```

Go to the project directory

```bash
  cd vue-tel-input
```

Install dependencies

```bash
  npm install
  cd docs && npm ci
```

Start the server

```bash
  npm run serve
```
